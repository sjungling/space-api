# Space API

This project is a chance to revisit front-end technologies and build a simple API for querying information about various space missions starting with the Apollo program.


## Getting Started
```shell
yarn install
yarn start
```


## Technology
- React 17
- Apollo Client 3
- Apollo Server
- Express
- Prettier
- TypeScript
- SQLite3
- Parcel
- Yarn
- TailWind 2
- Knex
